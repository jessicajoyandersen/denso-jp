🔵 Block C — RU-Verbs (Group 2)
Block C · 7–8 min · Drop る, add て

Block C
Group 2 — One Rule
Drop the final る, add て. Show the rule, then use it to unlock what they already knew.
⏱
Suggested time
7–8 min
💬 Talking Points
1
Write the rule: "miru → drop る → mi → add て → mite." Say it aloud as you write it. Then immediately: "mite kudasai — you've been saying this since Session 8. That's where it comes from."
2
Run through the conjugation table on the learner screen together — point at each row, have learners say the て-form before you confirm it.
3
Run the Build-It drill on the learner screen. Tap each verb — they say the て-form, then tap to confirm. Go twice around.
4
Phrase drill: call out a verb in -masu form (akemasu), learner converts to te kudasai form (akete kudasai). 5–6 rounds.
Quick Reference — Group 2 て-forms
mimasu→mitelook / check
tabemasu→tabeteeat
akemasu→aketeopen
shimemasu→shimeteclose
tsukemasu→tsuketeturn on / attach
misemasu→miseteshow
👁 Watch For
⚠️
tsukau vs tsukeru confusion — tsukau (use) is Group 1 → tsukatte. Tsukeru (turn on) is Group 2 → tsukete. Very different meanings and forms. Both appear in context — don't let them swap.
⚠️
Adding くだ before converting — some learners jump straight to "te kudasai" without converting the verb. Slow down if needed: verb first, then add kudasai.

📍 Block E — に (Destination)
Block E · 7–8 min · [destination] に itte/kite kudasai
5 / 6
Block E
に — Where You're Going
に marks a destination with movement verbs. The て-forms from Blocks C and D now get a location to move to.
⏱
Suggested time
7–8 min
💬 Talking Points
1
Pattern on the board: [destination] に itte kudasai. Point out that itte — the te-form of ikimasu — was previewed in Block D. Here it is in action.
2
Go through location words together: koko / soko / asoko mirror the kore/sore/are system from S6. Make the connection explicit: "Ko = near me, so = near you, a = over there. Same pattern."
3
Point at the capstone sentence on the learner screen: "Station 2 ni itte, kono tejun wo mite kudasai." Walk through each part: に (destination), itte (Group 1 irregular te-form), wo (S8 object particle), mite kudasai (Group 2 te-form + request).
4
Brief note on scope: "に does more than destination — it also marks where things exist (arimasu/imasu). That's Session 10. Today is destination only."
🔗 Ko/So/A Connection
S6 demonstratives → S9 location words. The same distance logic applies:
kore → koko  (near me)
sore → soko  (near you)
are → asoko (far from both)

If learners hesitate on koko/soko/asoko, remind them of kore/sore/are and the distance rule clicks.
👁 Watch For
⚠️
Dropping に — learners may say "Station san itte kudasai" without に. Gently correct: the particle is what marks the destination. Without it, the sentence is grammatically incomplete.
⚠️
itte vs kite confusion — itte = going there (away from speaker), kite = coming here (toward speaker). The speaker's position determines which one. Common mix-up in the two-verb sentences.
