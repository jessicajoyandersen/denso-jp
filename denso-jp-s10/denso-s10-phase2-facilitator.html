Block B
に — Where You're Going
One particle, any movement verb. Ties together te-form and location words from S6.
Talking points
1
"に after a place means you're going there. Station 3 ni itte kudasai — go to Station 3."
2
Walk through the capstone sentence: "Station 2 ni itte, kono tejun wo mite kudasai." Point out each particle — ni, wo, te-form chain.
3
Run the three floor exchanges as pair drills. Swap roles. Use 🔊 if pronunciation is uncertain.
Watch for
⚠️
itte vs iku: ikimasu → itte is a Group 1 exception. Not iite. Flag it if it comes up — U-verb rules in Block C will help explain.

Block C — U-Verbs

Block C
Group 1 — 4 Te-Form Patterns
The largest group. Pattern determined by the verb's final sound.
The 4 patterns
PatternRuleVerbs in widget
P1-u/-tsu/-ru* → -ttekau (buy), katsu (win), shiru (know)
P2-nu/-bu/-mu → -ndenomu (drink), tobu (fly)
P3-ku → ite, -gu → ide→ ite/idekaku (write), oyogu (swim)
P4-su → -shitehanasu (speak), kasu (lend)
Widget features
1
Each pattern is a separate tab. Green dot appears when all verbs in that pattern are complete.
2
Enter key checks each row. Correct answer locks and shows full conjugation as mora boxes (hiragana + romaji) to the right.
3
Part 2 has 8 MC items. Students must complete Part 1 before Part 2 unlocks.
Watch for
⚠️
shiru / kiru / kaeru: end in -ru but are U-verbs (Pattern 1 ru*), not RU-verbs. Students from S9 may default to dropping -ru — catch it early.
⚠️
iku → itte: fixed exception to Pattern 3, not iite. Already came up in Block B.
⚠️
oyogu → oyoide: Pattern 3 voiced variant uses -ide not -ite. Students sometimes write oyoite.

Block D
て-imasu — Ongoing Actions
They saw this in S9 Slide 4. Now it lands properly.
Talking points
1
"Remember the preview slide from last session? Te-form + imasu = what's happening right now. Today you learn it properly."
2
The key contrast to drill: "Tsukatte kudasai vs tsukatte imasu — same te-form, completely different meaning. Second word changes everything."
3
Point out that て-imasu uses all three groups — the examples on their screens show RU-verb, irregular, and U-verb forms.
4
MC item 4 is the key trap: "Tejun wo ___ kudasai" — they must choose mite kudasai (request) not mite imasu (ongoing). Make sure to debrief this one explicitly.
