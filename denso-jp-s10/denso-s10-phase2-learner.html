に — Where You're Going
に marks the destination with movement verbs. One particle, any movement verb.
The Pattern
[Destination] に [movement verb-te] kudasai

Station 3 ni itte kudasai. → Please go to Station 3.
Koko ni kite kudasai. → Please come here.

Itte = te-form of ikimasu (Group 1 exception: iku → itte, not iite). Kite = te-form of kuru (Group 3 irregular).
に in Non-Command Sentences
に marks the destination any time someone moves toward a place — not just in requests. You'll hear these on the floor every day.
Ie ni kaerimasu.
家に帰ります。
I'm going home. — plain statement, no command
Mainichi kaisha ni kite, shigoto wo shimasu.
毎日会社に来て、仕事をします。
I come to work every day and do my job. — に inside a chained te-form sentence
Kyuukei-shitsu ni ikimashita.
休憩室に行きました。
I went to the break room. — past tense, reporting where you went
The common thread
In all these sentences — command or not — に comes right after the destination. The verb that follows tells you the tense and whether it's a request. に itself always means the same thing: that's where they're headed.

Worker

Supervisor calls a worker over.
Supervisor
Chotto kite kudasai.
ちょっと来てください。
Please come here a moment.
Worker
Hai, tadaima.
はい、ただいま。
Yes, right away.
💡 Tadaima = right away / coming. Kite = te-form of kuru.
Line leader sends a worker to check a procedure at another station.
Line Leader
Station san ni itte, kono tejun wo mite kudasai.
ステーション3に行って、この手順を見てください。
Please go to Station 3 and check this procedure.
Worker
Hai, wakarimashita. Yoroshiku onegaishimasu.
Yes, understood. I'll take care of it.
💡 Two te-forms chained: itte (go) + mite kudasai (check). に = destination, を = object.

て-imasu — What's Happening Right Now
You saw this in Session 9. Now you learn it properly.
The Pattern
[verb-て] + います (imasu)

Describes an action that is currently happening or a state that is ongoing.

Kikai wo tsukatte imasu. → I'm using the machine (right now).
Kakunin wo shite imasu. → I'm doing the check.
Mite imasu. → I'm looking at it.
Floor Examples — te-form highlighted
Request
suwatte kudasai
Please sit down.
Ongoing
suwatte imasu
I'm sitting. / They're sitting.
suwaru (U-verb, P1) → suwatte
Request
mite kudasai
Please look / check.
Ongoing
mite imasu
I'm looking at it.
miru (RU-verb) → mite
Request
shite kudasai
Please do it.
Ongoing
shite imasu
I'm doing it. / It's being done.
suru (irregular) → shite
Request
kakunin shite kudasai
Please check / verify.
Ongoing
kakunin shite imasu
I'm checking / verifying.
kakunin suru → shite (suru compound verb)
て-imasu vs て-kudasai
Tsukatte kudasai → Please use it. (request — te + kudasai)
Tsukatte imasu → I'm using it. (ongoing — te + imasu)

Same te-form, different second word, completely different meaning. The te-form is the connector.
